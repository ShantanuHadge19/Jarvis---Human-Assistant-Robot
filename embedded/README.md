# Embedded Systems

This folder contains ESP32 code used in the JARVIS robot.

---

## Overview

The ESP32 is responsible for:
- Controlling all motors of the robot  
- Handling communication with the Raspberry Pi  
- Reading data sent by the Voice Module for command execution

---

## Files

- esp32_control.ino  
  Receives data from the voice module and Raspberry Pi and controls the motors accordingly.
