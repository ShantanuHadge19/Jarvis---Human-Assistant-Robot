# ROS2 Nodes

This folder contains individual ROS2 nodes used in the JARVIS robot.

## Nodes

- jarvis_core.py  
  Central node that handles communication and sends commands to the ESP32.

- idle_behavior.py  
  Manages robot behavior when no command is given.

- human_tracker.py  
  Tracks and follows a specific person.

- find_the_object.py  
  Detects objects based on user commands.

- automatic_docking.py  
  Handles autonomous docking for charging.
