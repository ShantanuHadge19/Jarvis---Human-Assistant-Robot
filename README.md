# Jarvis - Human Assistant Robot

ROS2-based human assistant robot integrating voice, gesture, and remote control, along with face tracking, object detection, and autonomous docking.

---

## Features

- Voice Controlled Robot
- Idle Mode for intelligent standby behavior
- Obstacle Avoidance
- Hand Gesture Control
- Human Tracking and Following
- Object Detection  
- Autonomous Docking for automatic charging
- Remote Control    

---

## Software and Hardware Used

- ROS2  
- Python
- Embedded C
- OpenCV  
- Raspberry Pi 4 
- ESP32  

---

## Project Structure

- `ros2_nodes/` → ROS2 functionalities 
- `embedded/` → ESP32 code  
- `hardware/` → Components and Design  
- `demo/` → Images and Videos  

---

## Note

This repository contains the final implementation modules of the project.
